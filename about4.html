  1. SMART home CONTROL  

(Completed and done with team)

An IOT based smart home controlling project. This is used to switch on and off the electric home appliances from anywhere in the world through the internet.

Description

            One single board computer (raspberry Pi) is used to control all the electric appliances and that is connected to internet. For the user side interface, a mobile application is developed. Additionally, user can also control using website. All these are connected to a cloud service called Firebase. Any action performed in user interface, firebase updates the status to the single board computer in the home which triggers the device accordingly.

Technologies Used: Java, JavaScript, HTML, CSS.

Languages: Python.
